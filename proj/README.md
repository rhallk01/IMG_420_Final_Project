# The Lunar Kingdom
**Final Project - 2D Platformer**
**Converted to C# and Updated**

---

## How to Run the Game
1.  Clone or download the project folder.
2. Open the project in Godot
3. Build and run the project 
4. On the main menu, press the **Start** button to begin the game

---

## Controls:
* Use the arrow keys or WASD and space to move
* Left key or A to go left
* Right key or D to go right
* Up key, space, or W to jump
* 
---

## Game Rules
### Win Conditions: 
* Collect the **key**
* Open the **door** while holding the key to win
  
### Lose Conditions:
* Touching the **enemy** will kill you, causing a Game Over
* Falling off the map will kill you, causing Game Over
